Протокол обмена:

Обмен инициируется клиентом (андроид, rpi).
1. Послать однобайтовую команду
2. Ждать ответа (тот же байт, что был послан) в течение 1 секунды
3. Если ответ не пришел или ответ неправильный, повторить (или ошибка)
4. Далее идет обмен данными в зависимости от команды
5. Данные (параметры и результаты) передаются в ASCII и заканчиваются переводом строки (0x0A)

?? Возможно, на время выполнения команды должен выключаться полив.

Команды.

V - получить версию ПО
-> V
<- V0.1\n

S - установить часы
-> S
<- S
-> 2014-06-26 22:58:00\n

G - получить показания часов
-> G
<- G2014-06-26 22:58:00\n

D - получить показание датчика
-> D
<- D
-> <номер датчика>\n
<- <значение>\n

L - получить расписание полива
-> L
<- L<строка;...;строка>\n

T - загрузить расписание полива
-> T
<- T<строка;...;строка>\n
